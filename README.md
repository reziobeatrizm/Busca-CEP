## 📌 Sobre o Projeto

Sistema de busca de CEP que retorna informações de endereço de forma rápida e prática, utilizando integração com API e envio de dados via Formspree.

---

## 🛠️ Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  
- API de CEP  
- Formspree  

---

## ⚙️ Funcionalidades

- 🔎 Consulta de CEP  
- 📍 Exibição de endereço completo  
- 📩 Integração com formulário (Formspree)  
- ⚡ Interface simples e intuitiva  

---

## 🎯 Objetivo

- Praticar consumo de APIs  
- Trabalhar com integração de formulários  
- Melhorar a manipulação do DOM  
- Evoluir habilidades em desenvolvimento front-end
